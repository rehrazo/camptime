# Camptime E-commerce Platform

Welcome to Camptime, an e-commerce platform for camping gear and equipment.

## Tech Stack
- Frontend: Vue.js 3
- Backend: Node.js/Express
- Database: MySQL
- Payments: Stripe
- Styling: Flex CSS

## Features
- Product Catalog
- Shopping Cart
- User Authentication
- Payment Processing with Stripe
- Order Management
